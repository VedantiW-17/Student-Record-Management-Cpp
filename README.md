# Student Record Management System (C++)

This is a simple Student Record Management System built using C++. It allows you to:

- Add student details
- Display all student records
- Search for a student by roll number
- Modify student details
- Delete a student record

## How to Run

1. Open a C++ compiler or IDE (e.g., CodeBlocks, DevC++, or online compilers).
2. Copy and paste the code from `student.cpp`
3. Compile and run the program.

## Features

- Uses array-based storage (no vectors).
- Menu-driven approach.
- Simple and clean C++ implementation with class-based design.
